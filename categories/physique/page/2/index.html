La relation entre ces deux grandeurs est : $$ \lambda= \dfrac{c}{\nu} $$ où $c$ est la célérité de l’onde dans le milieu dans lequel elle se propage ($c = \pu{3,0e8 m.s-1}$ dans le vide).

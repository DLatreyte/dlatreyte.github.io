 Posté le October 10, 2020  |  4 minutes  |  646 mots  |  David Latreyte

Courant électrique On appelle courant électrique le déplacement d’ensemble de charges électriques. Dans les solides, les charges mobiles sont les électrons ; Dans les solutions ionique, les charges mobiles sont les ions. Les électrons ne peuvent pas se déplacer librement dans les solutions ! À l’extérieur d’un générateur, le courant électrique circule du point de plus haut potentiel électrique (borne $+$ du générateur) vers le point de plus bas potentiel électrique (borne $-$ du générateur).
